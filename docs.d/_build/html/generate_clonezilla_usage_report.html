generate_clonezilla_usage_report module¶

Create a report of clonezilla images by querying mdb.

generate_clonezilla_usage_report.clonezilla_image_usage_report()¶

Collect the usage of clonezilla images from mdb as determined by a specific duty, usually tbs-server. Take the results of this information so it can be written to a filer.

generate_clonezilla_usage_report.get_clonezilla_usage_from_mdb(duty)¶

Query mdb for clonezilla images and retrieve the occurances of a given element in a list

generate_clonezilla_usage_report.numerically_sort_dictionary(dictionary_to_sort)¶

Sort a dictionary numerically.

generate_clonezilla_usage_report.run_a_command(cmd_to_run)¶

Run a standard linux command on a system.

generate_clonezilla_usage_report.write_data_to_csv(output_folder, dictionary)¶

Clonezilla uses 2 headers, image and count. Write the contents of a dictionary to a csv and put the headers at the top.
